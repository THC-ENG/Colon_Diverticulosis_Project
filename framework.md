##### **用少量专家标签把 MedSAM 调成教师模型，用它扩充伪标签和软知识，再把这些知识迁移到一个频域解耦、边缘敏感、推理高效的 ResSwinUNet 学生模型中，从而在低标注预算条件下实现准确且快速的结肠镜息肉自动分割**


**研究背景与意义**
---

结肠镜图像中的息肉分割是消化内镜智能辅助诊疗中的核心环节，其精确性直接影响病灶定位、边界判断和后续病理风险评估。然而，医学图像分割长期依赖专家逐像素标注，成本高、耗时长、主观性强，尤其在病灶边界模糊、形态多样、成像条件复杂时，构建高质量训练集更加困难。MedSAM 作为面向医学图像的基础分割模型，能够将通用的“Segment Anything”能力迁移到医学场景；其官方工作将其定义为一种“universal medical image segmentation”基础模型。与此同时，官方仓库还发布了 LiteMedSAM，并明确说明其推理速度约为 MedSAM 的 10 倍，侧面说明“基础模型能力强但计算开销大、实时部署受限”是一个真实存在的问题。

另一方面，结肠息肉分割领域已有较为成熟的公开数据基础，例如 Kvasir-SEG 官方提供 1000 张带像素级标注的息肉图像；PolypGen 是来自 6 个中心的多中心开放数据集，包含 1537 张 still images、2225 张 positive video sequences 和 4275 张 negative frames，适合用于跨中心泛化研究。经典公开测试集还包括 CVC-ClinicDB（612 张）、ETIS-LaribPolypDB（196 张）以及常见的 CVC-ColonDB / CVC-300 等外部评测数据。

本课题不再将“训练一个分割网络”作为唯一目标，而是围绕“如何在标注预算有限的前提下，借助基础模型减少专家标注负担，并将其能力蒸馏到可实时部署的学生网络中”这一核心问题展开。息肉数据在本课题中既是一个可复现实验载体，也是验证低标注预算分割框架可行性的标准平台。

###### **研究目标**

本课题拟构建一个低标注预算教师—学生分割框架，以少量专家标注驱动 MedSAM 的病灶域适配，再利用适配后的教师模型为大量未标注图像生成高质量伪标签，最终训练一个频域解耦、边界增强、推理高效的 ResSwinUNet 学生模型，实现对结肠镜息肉的准确、快速自动分割。具体目标如下：

1、构建一个规范的结肠息肉低标注预算实验协议，模拟“少量真实标注 + 大量无标注图像”的实际临床场景。

2、基于少量人工标注，采用 LoRA 参数高效微调 对 MedSAM 进行病灶域适配，降低全参数微调成本。LoRA 的核心思想是通过低秩增量实现参数高效适配，最初由 Hu 等人在 ICLR 2022 提出。

3、利用域适配后的 MedSAM 自动或半自动生成伪标签，形成“伪标签扩充—人工抽检—再训练”的 SAM-LoRA 数据飞轮。

4、在学生端构建频域解耦 ResSwinUNet，并引入轻量频域解耦模块与 logits-edge consistency 边缘一致性约束，以提升模型对小目标、模糊轮廓和细粒度边界的表征能力。

5、在训练优化层面引入 结构解耦 + 空间分工 + 分阶段训练 + 动态权重组合，作为多目标优化的验证性强化线，用于缓解分割损失、边界损失、蒸馏损失之间的梯度冲突。

6、系统评估该框架在精度、泛化能力与推理效率上的表现，验证其对“专家标注难”问题的缓解作用。

###### **拟解决的问题**

（1）标注稀缺问题

在医学内镜分割中，专家像素级标注代价高昂，导致高质量监督数据难以快速扩充。如何利用少量专家标签驱动基础模型生成可靠伪标签，是本课题的首要问题。

（2）基础模型与部署模型之间的矛盾

MedSAM 具备较强的医学分割先验，但其交互式特性和计算开销决定了它更适合作为教师模型或标注辅助工具，而非直接用于实时临床部署。如何将其能力迁移到更轻、更快的任务专用网络，是本课题的第二个问题。MedSAM 官方发布 LiteMedSAM 并强调其加速版本，正说明这一矛盾具有现实意义。

（3）多目标训练中的优化冲突问题

学生模型训练并非单一分割目标，而是同时包含主分割监督、边界监督、伪标签监督乃至蒸馏监督。多损失共同优化时常出现梯度方向冲突，导致某一目标主导训练或抑制其他目标。如何稳定训练过程、提升不同监督信号间的协同，是第三个问题。

**总体思路与技术路线**：少量真值驱动教师适配—大规模伪标签扩充—高效学生蒸馏学习，形成一个从数据到模型再到部署的闭环。

###### **创新点**

（1）提出面向低标注预算的 MedSAM—ResSwinUNet 教师学生框架

不再将 MedSAM 仅作为标注工具，而是将其作为具备医学分割先验的教师模型，用少量专家标签驱动域适配，再将其知识迁移到实时学生网络中。

（2）构建 SAM-LoRA 数据飞轮，面向“专家标注难”问题形成可持续扩充机制

通过“少量真值适配—大规模伪标签生成—质量筛选—反哺再训练”的闭环，降低新增病灶分割数据集的构建成本。

（3）引入轻量频域解耦模块

不采用大规模、全网络频域改造，而是通过小波分解显式区分低频语义信息与高频边界信息，以较低结构代价提升模型对模糊边界和细粒度纹理的建模能力。

（4）边缘一致性替代重型边界头

以更轻的 logits-edge consistency 作为监督侧，约束主分割结果边界，同时把原边界头降级为验证线，降低梯度冲突和结构复杂度。

（4）软硬结合监督

伪标签提供确定性区域监督，蒸馏提供边界和不确定性先验

###### **数据集构建**

采用联合数据集：Kvasir-SEG，CVC-ClinicDB，CVC-ColonDB，ETIS，CVC-300，PolypGen

1、第一层：教师/学生共享的训练宇宙

Kvasir-SEG：1000 张带像素级标注图像，作为基础监督源之一。

CVC-ClinicDB：612 张图像，作为第二个主训练源。

PolypGen训练中心：选择若干训练中心的 still images 及稀疏采样的 positive sequences，用于增强多中心泛化。PolypGen 来自 6 个中心，具有多中心、跨域优势。

在这堆数据里，再人为切成：

L\_small：少量有标注子集

U\_large：大量无标注子集

2、第二层：严格外部测试

ETIS-LaribPolypDB：196 张，作为高难外部测试集。

CVC-ColonDB / CVC-300：作为经典跨数据集测试集。关于其样本数，文献存在 300/380/60 等不同命名与统计口径，因此后续整理时应以具体下载源和文件清单核验，而非只看名称。

PolypGen 保留中心：整中心留出，用于评估跨中心泛化。

###### **论文总体框架**

阶段一：构建低标注预算数据协议

从 Kvasir-SEG、CVC-ClinicDB、PolypGen 等公开息肉数据中构建共享训练系统，其中仅保留一小部分为真实标注集 L\_small，其余作为无标注集 U\_large 使用；同时保留 ETIS、CVC-ColonDB、CVC-300 以及 PolypGen 的保留中心作为严格外部测试集，不参与任何调参，仅用于外部泛化评估。该数据协议能够较好模拟真实临床场景。

阶段二：基于 SAM-LoRA 的域适配、伪标签构建与数据飞轮

以 MedSAM 为教师，使用 L\_small 上的少量专家真值进行域适配。采用 LoRA 方式对图像编码器或关键注意力层进行参数高效微调，输出一个具备结肠息肉分割先验的教师模型。使用该模型对 U\_large 进行自动或半自动分割，生成第一轮伪标签；再通过置信度筛选、边界质量筛选和少量人工抽检清洗伪标签，将高质量伪标签重新加入训练池，形成“教师适配 → 自动标注 → 质量过滤 → 反哺训练”的数据飞轮。

阶段三：学生端频域解耦 ResSwinUNet 自动分割网络

学生端以现有的 ResSwinUNet 框架为主体，在 bottleneck 位置加入轻量频域解耦模块，通过二维离散小波变换将特征分解为低频语义分量与高频边界分量，分别由 Swin Transformer 主干和轻量高频精炼旁路并行处理，再经逆小波变换重构回空间域。在优化层面，本文以主分割监督和边缘一致性损失作为主线设计，增强边界建模能力；同时将 结构解耦 + 空间分工 + 分阶段训练 + 动态权重 作为验证性强化线，环节梯度冲突。

阶段四：教师—学生联合学习与快速自动分割

学生模型最终以真值标签和高质量伪标签联合训练，并进一步引入蒸馏监督，使其学习教师模型的软分割先验与结构知识。最终训练出的频域解耦 ResSwinUNet 将作为部署阶段的核心模型，实现对结肠镜息肉的快速自动分割。

###### **Wavelet-Decoupled 频域深度学习**

Encoder 输出 → DWT → 低频进 Swin，高频走轻量 Refiner → IDWT 重建 → Decoder

具体结构：（只先放在 bottleneck）

对进入 Swin 前的特征图 F 做一次二维 Haar DWT：F --> {LL, LH, HL, HH}

其中：

LL：低频语义子带，保留主体结构和大范围轮廓

LH/HL/HH：高频细节子带，主要承载边缘、方向性纹理和局部突变信息

然后：

低频支路：LL 直接送入原来的 Swin blocks，让自注意力主要处理低频结构信息和长距离依赖。这一步保留了现有 Swin 模块最擅长的东西：全局建模和上下文理解

高频支路：把 LH, HL, HH 在通道维拼接后，送入一个极轻量的 High-Frequency Refiner

最后重建：

将 refined high-frequency 与 low-frequency 一起经 IDWT 重建回空间域，再送给 decoder.
High-Frequency Refiner 结构：

1×1 Conv: 3C → C

GroupNorm + GELU

Depthwise Separable Conv 3×3

GroupNorm + GELU

1×1 Conv: C → 3C

Residual add

###### **方法设计**

**教师端：SAM-LoRA 域适配与伪标签生成**

教师端的核心目标不是“重新标注已有真值数据”，而是将少量专家标签压缩成可扩展的标注能力。具体设计如下：

使用少量真值图像对 MedSAM 进行 LoRA 微调，使其快速适应结肠镜息肉的颜色、纹理、边界和尺度分布。

在推理端，以框提示、点提示或由粗分割网络生成的候选区域作为 prompt，驱动 MedSAM 自动生成病灶初始 mask。

依据输出置信度、区域连通性、边界平滑度、面积阈值等规则过滤低质量伪标签。

对高置信伪标签直接纳入训练，对中等置信伪标签进行人工快速修正，对低置信样本回收至困难样本池。

通过多轮循环不断提高教师在目标域上的标注质量，形成 SAM-LoRA 数据飞轮。

这一部分目的在于：降低了专家逐像素画 mask 的成本，使专家只需修正少量高难样本，从而得到低成本流程。

**教师端：SAM-LoRA 数据飞轮**

Round 0：教师初始化

用 L\_small 对 MedSAM 做 LoRA 域适配,得到初始教师 T0.

Round 1：第一轮伪标签扩充​

用 T0 对U\_large 全量推理，按置信度、边界质量、区域合理性筛选，选取前 30%–40% 的高质量样本加入训练池。

Round 2：第二轮困难样本扩充

以第一轮训练后的教师或学生辅助重新处理剩余未标注样本，再选取额外 10%–20% 的高质量样本加入训练池。

停止条件：3 条满足其一即可停止：

1、外部验证集 Dice 提升小于预设阈值。例如 < 0.3 个百分点

2、新增伪标签的人工抽检合格率下降到阈值以下。例如 < 85%

3、第二轮后增益明显趋于饱和，不再继续扩充

**学生端：频域解耦 ResSwinUNet**

（1）主线：ResSwinUNet + 频域解耦 + 边缘一致性约束

ResNet 浅层编码器有利于稳定捕获局部纹理

Swin 模块有利于引入更强的全局依赖建模

Attention Gate 可以抑制跳跃连接中的背景噪声

在 bottleneck 插入一次二维 Haar DWT/IDWT，进行表征层面的边界增强

主分割头输出logits，对logits或sigmoid后的probability提取边缘，再让此边缘和GT边界对齐

频域模块和边缘一致性都服务于边界建模，但不重复，而是互补。

频域解耦模块负责特征表示层面，让高频边缘信息更容易被保留下来

边缘一致性约束负责监督约束层面，直接要求预测结果在边界位置更贴合 GT

频域模块负责“让网络看见边界”，边缘一致性负责“让网络学会边界

（2）验证线：结构解耦 + 空间解耦 + 时间解耦 + 动态权重

结构解耦：频域模块把低频语义和高频边缘拆开，这减少了主分割和边界学习挤在同一套特征上的冲突。

空间解耦：不同监督作用在不同区域。高置信区域为伪标签硬监督，中等置信 / 边界带为软蒸馏，GT 边界窄带为 edge consistency。这样避免多种损失在同一位置硬碰硬。

时间解耦：采用分阶段训练。先只训真实标签主分割，再加边缘一致性，再加伪标签监督，最后加蒸馏，这样避免训练初期多损失同时扰动 backbone。

权重调度：不是一开始所有损失同权重，而是 edge loss 小权重起步，pseudo loss 逐步升温，distill loss 最后再引入，且权重更小。

**蒸馏学习机制**

为进一步增强学生模型对教师模型知识的吸收能力，本课题在伪标签监督之外引入教师—学生蒸馏机制。蒸馏监督主要分为两个层次。

第一层为响应蒸馏，即利用教师模型输出的 soft mask 概率分布对学生模型进行约束，使学生不仅学习伪标签的“硬边界”，还能够学习教师输出中包含的区域置信度与软分割先验。

第二层为边缘蒸馏，即从教师输出中提取边缘响应，并约束学生模型学习教师在边界建模方面的表达能力。该部分与边缘一致性约束具有天然协同性，有助于提升学生在复杂轮廓和模糊过渡带上的分割稳定性。通过将蒸馏监督与真实标签监督、伪标签监督联合优化，本文旨在使学生模型不仅学习“伪标签的硬边界”，还能够学习教师模型更平滑、更具先验性的软分割知识，从而在有限标注条件下进一步提升分割性能与泛化能力。

###### **训练策略设计**

1、学生模型训练时包含三类监督：

真实标签监督：对 L\_small 使用标准分割损失，保证训练不偏离医学标注标准

伪标签监督：对高质量伪标签使用降权监督，提升数据利用率

蒸馏监督：让学生拟合教师的 soft mask、边界响应或中间特征表征，提升学生对教师先验知识的吸收程度

2、可将总损失写为：L\_total​=λ\_1\*L\_sup​+λ\_2\*L\_pseudo​+λ\_3\*L\_edge​+λ\_4\*L\_distill​

L\_sup​：真实标签上的监督分割损失，可由 Dice + Focal/BCE 构成

L\_pseudo：伪标签监督损失，可按伪标签置信度加权

L\_edge​：边缘一致性损失，通过对分割 logits 提取边缘响应并与 GT 边界进行约束

L\_distill：蒸馏损失，用于约束学生学习教师的软分割先验，可采用 soft BCE、KL 或 feature alignment，分为响应蒸馏和边缘蒸馏



**\*边缘一致性约束（logits-edge consistency）**是指：不额外引入显式边界预测头，而是直接从主分割分支输出的概率图中通过 Sobel 等梯度算子提取预测边缘，并与由真实 mask 生成的边界图进行一致性约束。该策略能够在较低结构开销下增强模型对病灶轮廓和过渡带区域的敏感性，从而提升模糊边界场景下的分割稳定性。
